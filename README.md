A modern, responsive e-commerce storefront built with Next.js and TypeScript.  
This project provides a clean, component-driven UI for browsing products, managing a shopping cart, and handling user authentication flows.

---

## Features
- Product listing and details pages
- Cart management with quantity controls and coupons
- User authentication pages: signup, login, OTP, reset password
- Responsive layout and accessible components
- Component-driven architecture (atoms, molecules, organisms)
- Local mock data support for rapid development

---

## Tech Stack
- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS / PostCSS**
- **React Context** for state (Cart + Products)
- **Framer Motion** for animations
